# Logger

This is my own implementation of the ESP-IDF logger, adjusted to fit my likings a little.

### Implements:
* `LOGE(tag, format, ...)` - Logs using the error log level
* `LOGW(tag, format, ...)` - Logs using the warning log level
* `LOGI(tag, format, ...)` - Logs using the info log level
* `LOGD(tag, format, ...)` - Logs using the debug log level
* `LOGV(tag, format, ...)` - Logs using the verbose log level
