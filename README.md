# ESP-IDF-Library

This repository contains all my custom ESP-IDF libraries I've written so far.

> [!IMPORTANT]  
> The libraries are all written in C++, so you will likely need to write your main in C++ too.
> This can be done using `extern "C"` infront of your `app_main()` declaration.

### Folder Structure:
* `general` - Contains general-purpose libraries useful for nearly all projects
